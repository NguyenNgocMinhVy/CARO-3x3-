## System Overview
The system is a simple Caro 3x3 (Tic Tac Toe) game developed using C# WinForms.

## Stakeholders
- Player
- Developer

## Functional Requirements
1. Player can click a cell.
2. System places X.
3. Computer places O.
4. System checks winner.
5. System highlights the winning line.

## Non-Functional Requirements
- Fast response
- Simple UI
- Runs on Windows
