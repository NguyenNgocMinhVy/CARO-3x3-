# Caro 3x3 Game

A simple **Caro (Tic Tac Toe) 3x3** game built with **C# WinForms**.

## Features

* Player vs Computer
* Highlight winning line
* Simple and responsive UI

## Technologies

* C#
* .NET WinForms
* Visual Studio

## How to Run

1. Open the project in **Visual Studio**
2. Build and run the solution
3. Play the game
